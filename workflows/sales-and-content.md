# Workflow: Sales Pipeline

**Pipeline:** Hermes → Nike → Peitho → Doros → Tyche

Use this when launching sales for a new product or entering a new market segment.

---

## Steps

### Step 1: Brief (Hermes)
```
"Hermes, write a sales brief for [product] targeting [segment]"
```

### Step 2: Sales Strategy (Nike)
```
"Nike, build a sales strategy from this brief: [link]"
```
Nike defines ICP, channel mix, and pipeline targets.

### Step 3: Sales Copy (Peitho)
```
"Peitho, write outreach sequences from Nike's strategy: [link]"
```
Peitho produces cold emails, LinkedIn sequences, and a proposal template.

### Step 4: Demo Materials (Doros)
```
"Doros, create a pitch deck and demo script from this strategy: [link]"
```

### Step 5: CRM Setup (Tyche)
```
"Tyche, set up pipeline stages and forecasting based on Nike's strategy: [link]"
```

# Workflow: Content Campaign

**Pipeline:** Prometheus → Apollo → Calliope → Orpheus → Eos

Use this for a coordinated content push: launch, campaign, or thought leadership series.

---

## Steps

### Step 1: Campaign Strategy (Prometheus)
```
"Prometheus, define a content campaign strategy for [goal/topic]"
```

### Step 2: Long-form Content (Apollo)
```
"Apollo, write [blog post/whitepaper] from this strategy: [link]"
```

### Step 3: Conversion Copy (Calliope)
```
"Calliope, write the landing page and email copy for this campaign: [link]"
```

### Step 4: Social Visuals (Orpheus)
```
"Orpheus, design social media visuals for this campaign: [link]"
```

### Step 5: Publishing Schedule (Eos)
```
"Eos, build a content calendar and posting schedule for this campaign: [link]"
```
